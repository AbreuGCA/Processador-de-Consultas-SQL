# Projeto de Álgebra Relacional

Este projeto implementa operações de álgebra relacional em Python, com suporte a análise de grafos, parsing de comandos e validação de esquemas.

## Estrutura dos Arquivos
- `algebra_relacional.py`: Operações de álgebra relacional.
- `app.py`: Ponto de entrada principal ou interface do projeto.
- `dic_dados.py`: Dicionário de dados ou manipulação de metadados.
- `grafo.py`: Funções para análise e manipulação de grafos.
- `parse.py`: Parsing de comandos ou expressões.
- `val_schema.py`: Validação de esquemas relacionais.

## Como usar
1. Crie e ative um ambiente virtual:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Linux/macOS
   .venv\Scripts\activate    # Windows
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute o projeto conforme instruções no `app.py` ou documentação interna dos arquivos.
